# Leitura de Arquivos em Java (Projeto de Estudo)

Este é um projeto simples de estudo em Java, criado para praticar a manipulação básica de arquivos.

## Descrição

O programa lê um arquivo de texto linha por linha utilizando a classe `Scanner` e imprime seu conteúdo no console.

Este projeto faz parte do meu processo de aprendizagem em Java, e o código provavelmente será modificado e aprimorado no futuro.

## Tecnologias

- Java (JDK 21 ou compatível)
- Scanner
- Entrada e saída de arquivos (File I/O)

## Como executar

1. Coloque um arquivo de texto chamado `in.txt` na raiz do projeto.
2. Execute a classe `Program`.
3. O conteúdo do arquivo será exibido no console.
